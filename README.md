# International Student Housing Finder

A sample project repository for planning and documenting a solution that helps international students in Australia find suitable housing more easily.

## Project Overview
International students often struggle to find accommodation that is affordable, safe, and conveniently located near their campus. This project explores a platform concept that helps students compare housing options, understand the local market, and make better accommodation decisions.
